HTML Document Structure

HTML documents have a basic structure that includes elements like DOCTYPE, html, head, and body.

            <!DOCTYPE html>
            <html>
            <head>
                <title>Document Title</title>
            </head>
            <body>
                Content goes here
            </body>
            </html>
