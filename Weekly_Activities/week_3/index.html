What is External CSS?

External CSS involves creating a separate CSS file and linking it to your HTML document. This approach offers several advantages:

  • Separation of content (HTML) and presentation (CSS)
  • Reusability across multiple pages
  • Easier maintenance
  • Faster page loading (browser caching)
  • Cleaner code organization
<!-- How to link to external CSS -->
<head>
  <link rel="stylesheet" href="css/styles.css">
</head>

Separation of Concerns

External CSS allows you to separate your content (HTML) from its presentation (CSS). This makes your code more maintainable and easier to understand.

Common CSS Properties

CSS includes a wide range of properties to control the appearance of your web pages. Here are some of the most commonly used categories:

Category Examples Description
Text Properties color, font-family, font-size, text-align Control the appearance and layout of text
Box Model margin, padding, border, width, height Control the spacing and dimensions of elements
Background background-color, background-image Control the background appearance of elements
Layout display, position, float, flex, grid Control how elements are positioned on the page

Visit the CSS Properties page to learn more about these properties and how to use them effectively.
